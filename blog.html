Discuss the scope of var, let, and const


Var : A var can be declared anywhere within a JavaScript file

let : let declarations are scoped to blocks as well as functions

const : const variables can neither be updated nor re-declared.

Tell us the use cases of null and undefined


Null : Null means an empty value or a blank value.

Undefined : Undefined means the variable has been declared, but its value has not been assigned.
